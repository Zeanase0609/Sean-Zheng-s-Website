10DGT Learning Journey

Welcome to my 10DGT learning journey! Here, I will share my experiences and projects related to programming, including things like image editing, programming with Python and web development with HTML/CSS.

Image Editing

Image editing is the process of creating and designing characters and setting. It includes tasks like planning, designing, and creating designs.

Programming

Programming is the process of writing code to create software applications. It involves using programming languages like Python and HTML/CSS to build functional and interactive programs. For programming, we learnt how to write code in python, as well as creating websites using HTML/CSS.

3D Modelling

3D modelling is the process of creating three-dimensional objects using specialized software. It involves designing, sculpting, and texturing models for use in various applications such as video games, animations, and simulations. We learnt to create 3D models using Blender.

Animation

Animation is the process of creating moving images through the manipulation of static images. It involves techniques like frame-by-frame animation, rigging, and keyframing. We explored animation using software like Blender.

Video Game Development

Video game development is the process of creating and designing video games. It involves various stages such as concept development, design, programming, and testing. We explored game development using tools like GDevelop and itch.io.
